Регистр сведений предназначен для примера использования подсистемы Даты запрета изменения.

В подсистеме Даты запрета изменения регистр сведений показывает пример запрета изменения записи по разделу Демо: Складской учет с объектом Демо: Место хранения. При этом в качестве проверяемой даты используется дата из поля Период.
