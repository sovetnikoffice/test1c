Документ предназначен для примера использования подсистем Даты запрета изменения, Префиксация объектов, Управление доступом.

В подсистеме Даты запрета изменения документ показывает пример запрета изменения документа по разделу дат запрета Демо: Складской учет с двумя объектами Демо: Место хранения, которые получаются из двух реквизитов Место хранения источник и Место хранения приемник.

В подсистеме Префиксация объектов к номеру документа добавляются префиксы организации и информационной базы.

В подсистеме Управление доступом документ показывает типичный пример логики ограничения для документов перемещения.
В данном случае логика ограничения для права Чтения - это разрешенность любого из мест хранения, а для права Изменения - это разрешенность одновременно двух мест хранения.
Документ записывает наборы значений доступа для журнала документов Демо: Складские документы.
