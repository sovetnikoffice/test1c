Документ предназначен для примера использования подсистем Даты запрета изменения, Префиксация объектов, Управление доступом.

В подсистеме Даты запрета изменения документ показывает типичный пример запрета изменения документа по разделу дат запрета Демо: Складской учет с объектом Демо: Место хранения.

В подсистеме Префиксация объектов к номеру документа добавляются префиксы организации и информационной базы.

В подсистеме Управление доступом документ показывает пример ограничения по Роли-дополнительному праву Демо: Печать непроведенных документов. Для проверки работы печати непроведенных документов предусмотрена команда Демо: Проверка разрешения на печать непроведенных документов.
Документ записывает наборы значений доступа для журнала документов Демо: Складские документы.
