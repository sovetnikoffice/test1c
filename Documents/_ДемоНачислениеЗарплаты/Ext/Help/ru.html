Документ предназначен для примера использования подсистемы Управление доступом.

В подсистеме Управление доступом документ показывает пример реализации ограничения доступа требующее разрешенности всех значений в табличной части одновременно (а не какого-нибудь одного). В данном случае требуется разрешенность всех физических лиц, заданных в табличной части Зарплата. Такое ограничение может быть реализовано только на основе шаблона #ПоНаборамЗначений. Для реализации требуется создать табличную часть НаборыЗначенийДоступа в объекте - документе Демо: Зарплата.
