Документ предназначен для примера использования подсистем Даты запрета изменения, Дополнительные отчеты и обработки, Печать, Префиксация объектов, Управление доступом.

В подсистеме Даты запрета изменения документ показывает типичный пример запрета изменения документа по разделу дат запрета Демо: Складской учет с объектом Демо: Место хранения.

В подсистеме Дополнительные отчеты и обработки документ используются в качестве примера того, как прикреплять к объектам конфигурации команды формирования внешних печатных форм.

В подсистеме Печать документ показывает пример реализации печатной формы, размещенной в обработке _ДемоПечатнаяФорма (это устаревший метод - для иллюстрации обратной совместимости; для штатного способа внедрения см. документ Демо: Счет на оплату покупателю)

В подсистеме Префиксация объектов к номеру документа добавляются префиксы организации и информационной базы.

В подсистеме Управление доступом документ показывает пример использования различной логики ограничения для прав Чтение и Изменение.
Документ записывает наборы значений доступа для журнала документов Демо: Складские документы.
