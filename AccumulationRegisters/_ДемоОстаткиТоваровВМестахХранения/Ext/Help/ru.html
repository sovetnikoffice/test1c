Регистр накопления предназначен для предназначен для демонстрации использования подсистем:

  • Управление доступом - показывает пример логики ограничения регистров по измерениям, ограничение по организации и месту хранения с использованием шаблона ограничения доступа #ПоЗначениям.
  • Базовая функциональность - корректность замены ссылок, поиска и удаления дублей.
