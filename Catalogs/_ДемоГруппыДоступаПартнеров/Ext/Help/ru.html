Справочник предназначен для примера использования в подсистемах Даты запрета изменения и Управление Доступом.

В подсистеме Даты запрета изменения элементы справочника используются, как "разрез" дат запрета изменения.
Элементы справочника - объекты раздела дат запрета изменения Торговля.

В подсистеме Управление доступом справочник Демо: Группы доступа партнеров используется в качестве списка групп доступа значений доступа.
        Значения доступа - это партнеры, а группы доступа - элементы справочника Демо: Группы доступа партнеров (легко перепутать со справочником Группы доступа, который имеет другой смысл).
        Элементы справочника Демо: Группа доступа партнеров используется вместо значений доступа Демо: Партнеры в справочнике Группы доступа, при редактировании списка разрешенных значений доступа.
