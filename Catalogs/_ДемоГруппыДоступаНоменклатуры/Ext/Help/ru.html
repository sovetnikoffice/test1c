Справочник предназначен для примера использования в подсистеме Управление Доступом.

В подсистеме Управление доступом справочник Демо: Группы доступа номенклатуры используется в качестве списка групп доступа значений доступа.
        Значения доступа - это номенклатура и виды номенклатуры, а группы доступа - элементы справочника Демо: Группы доступа номенклатуры (легко перепутать со справочником Группы доступа, который имеет другой смысл).
        Элементы справочника Демо: Группа доступа номенклатуры используется вместо значений доступа Демо: Номенклатура и Демо: Виды номенклатуры в справочнике Группы доступа, при редактировании списка разрешенных значений доступа.
        В данном примере один справочник Демо: Группа доступа номенклатуры используется для двух типов значений доступа _Демо: Номенклатура и _Демо: Виды номенклатуры.
